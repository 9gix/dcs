# CS2102: Digital Content Store

Digital Content Store Project (Semester 2, 2014/2015).

## Documentation

[ER Models][erd]

## Setup

```

# Install requirement
pip install -r requirements.txt

# Migrate Database
python manage.py migrate

# Create Super USer
python manage.py createsuperuser

# Start Local Server
python manage.py runserver
```
